# Vue Pattern Lab

Vue 기반 프론트엔드 개발 패턴을 정리한 개인 아카이브입니다.

이 저장소는 단순 코드 모음이 아닌,
- 화면 구조 설계
- 상태 관리 타이밍
- 인증/권한 처리
- API 연동 방식

등 **실제 서비스 개발에서 반복적으로 사용된 패턴**을 중심으로 정리합니다.

## Structure

- components/
  - 바로 사용 가능한 컴포넌트
- patterns/
  - 재사용 가능한 UI / 상태 / API / 인증 패턴 정리
- examples/
  - 패턴이 실제로 적용된 코드 예시
- dev-notes/
  - 리팩토링 기록, 시행착오, 개인 메모

-- 추후 변경될 수 있음. 고민중

## Goal

- 다음 프로젝트에서 그대로 복붙 가능한 구조 확보
- 프론트엔드 설계 사고방식 축적
